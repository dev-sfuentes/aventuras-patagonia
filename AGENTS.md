# Project Context

## Summary

This project is an informative website for a Chilean PyME focused on horseback riding experiences.

## Product Scope

- Informational website only
- No backend
- No user accounts
- No server-side business logic beyond what Nuxt 3 needs for the frontend build/runtime

## Audience And Language

- Primary website language: English
- Secondary website language: Spanish
- Business location: Chile, Punta Arenas

## Technical Stack

- Frontend framework: Nuxt 3
- Development environment: `devenv`
- JavaScript runtime/package manager available in development: Bun

## Deployment And Repositories

- Primary repository location: GitLab
- Mirrored/synced repository: GitHub
- Hosting platform: Cloudflare Pages

## Working Assumptions

- The site should prioritize a clear marketing/information architecture over application-style complexity.
- Internationalization should be considered from the start, with English first and Spanish second.
